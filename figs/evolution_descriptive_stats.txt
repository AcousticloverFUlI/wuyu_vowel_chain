- N/T 同进退：61/64 点；其中 N/T 均合流 34 点，N/T 均分立 27 点。

二、强蕴涵关系
- 五个核心声母全分立 => N/T 也均分立: 3/3，置信度 100.0%；反例：无
- 方言点内只要出现 S0=S1 => 同点也出现 S2=S3: 30/30，置信度 100.0%；反例：无
- M 组达到 S1=S2=S3 => 五个核心声母均已有某种合流: 19/19，置信度 100.0%；反例：无
- S2=S3 扩展到至少四个核心声母 => N/T 均合流: 31/31，置信度 100.0%；反例：无
- 出现 S1=S3 越级 => 同点也出现 S2=S3: 2/2，置信度 100.0%；反例：无
注意：S1=S3 越级只有 2 点，支持量低，只能作为边缘现象记录。

三、高概率蕴涵关系
- K 组出现 S0=S1，S2=S3 断裂双合并 => N/T 多数均合流: 25/26，置信度 96.2%；反例：湖州市区
- N/T 均合流 => 五个核心声母多数也全有合流: 32/34，置信度 94.1%；反例：富阳城关、江阴城区澄江镇
- 出现 S0=S1 => 多数已经不是局部合流，而是核心声母广泛合流: 28/30，置信度 93.3%；反例：丹阳、湖州市区
- 出现 S1=S2 => 多数同点也出现 S2=S3: 22/26，置信度 84.6%；反例：上虞百官镇、嵊州崇仁镇、柯桥城关、绍兴城区东头埭
- 五个核心声母全有合流 => 多数 N/T 也均合流: 32/40，置信度 80.0%；反例：上虞百官镇、安吉安城、嵊州崇仁镇、柯桥城关、绍兴城区东头埭、萧山南阳镇、萧山城厢、象山丹城

四、关键反例与分支解释
- 五个核心声母全有合流但 N/T 没有均合流的点：上虞百官镇、安吉安城、嵊州崇仁镇、柯桥城关、绍兴城区东头埭、萧山南阳镇、萧山城厢、象山丹城。
  这些点说明“核心声母全合流”本身有两条路：一条是 S2=S3 向外扩展的高端合流路，另一条是 S1=S2 较突出的中层合流路；后者不一定带动 N/T。
- N/T 不同进退的点：安吉安城、郎溪梅渚镇定埠、靖江城区。
  N/T 总体高度同向，但这几个点提示鼻音、塞音条件仍可能有局部差异。

五、类型学解释
1. S2=S3 是最基础的合流层。只要一个方言点已经出现 S0=S1，它一定也已经在某处出现 S2=S3，因此低端合流不是独立起步，而是建立在高端合流之后。
2. K 组的 S0=S1，S2=S3 是“高级阶段”信号。它几乎总是伴随 N/T 均合流，说明 K 组一旦发生断裂双合并，往往已经进入系统性重组。
3. M 组的 S1=S2=S3 是“核心声母全面合流”信号。它不必然带动 N/T，但一旦出现，五个核心声母通常都不再保持全分立。
4. N/T 均合流比单个核心声母合流更像系统阈值。尤其当 S2=S3 扩展到四个以上核心声母时，N/T 均合流没有反例。
5. 因此点级演变可以粗略理解为：全分立 -> 局部 S2=S3 -> 多声母 S2=S3 -> K 组断裂双合并或 M 组连续三合一 -> N/T 也进入合流。但临绍、萧山、象山等点显示出另一条以 S1=S2 为中心的分支，不一定同步带动 N/T。
